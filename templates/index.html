How it works
  1. Enter a document URL and your Jina API key
  2. The application will extract content from the document using Jina's Reader API
  3. The content will be segmented into chunks using Jina's Segmenter API
  4. Embeddings will be generated for the chunks using Jina's Embeddings API
  5. You can then ask questions about the document
  6. The application will find the most relevant chunks to answer your questions
